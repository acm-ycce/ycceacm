Cyber Awareness Program

The Cyber Awareness Program event, coordinated by Prof. Supriya Thombre and Prof. Nikhil Mangrulkar, was organized by the Computer Science and Engineering department at YCCE on 4 January 2020. Overall, the Cyber Awareness Program event was a great success, providing participants with a valuable learning experience and networking opportunities. The event was a testament to the department's commitment to promoting cybersecurity awareness and excellence in the field of computer science and engineering.

Object Oriented Programming Using Java

The two-day workshop on Java programming language was organized by ACM in YCCE. On the first day, Ms. Aditi Kale introduced the history and features of Java, followed by Mr. Suraj Nair who covered programming concepts such as variables, data types, and control structures. Mr. Yash Waghmare then explained the class and object concept.
On the second day, Ms. Purva Natoo covered the implementation of object-oriented concepts like encapsulation and inheritance with programming examples. Mr. Saurabh Charade then explained topics such as abstraction and polymorphism, with a focus on runtime polymorphism, and gave plenty of examples. The session concluded with a small assignment given to students, and both HoD and Deshkar madam expressed appreciation for the alumni speakers and thanked them for their time.
